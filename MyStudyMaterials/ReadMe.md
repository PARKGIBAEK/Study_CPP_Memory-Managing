C++ IOCP 채팅 서버 프로젝트

이 프로젝트의 목적은 C++ MMORPG 게임 서버의 코어 라이브러리 중 기본적인 것들을 배우는 것이다.



최종 완성 프로젝트의 구성

1. ServerCore(코어 라이브러리)
   - IocpCore
   - Session
   - Service
   - SendBuffer
   - RecvBuffer
   - NetAddress
   - MemoryPool
   - ObjectPool
   - Listener
   - SocketUtils
   - ThreadManager
   - Lock
   - DeadLockProfiler
   - BufferWriter
   - BufferReader
   - JobTimer
   - JobQueue
   - GlobalQueue
2. GameServer(서버)
3. DummyClient(더미 테스트 용)
